# IPTV Indonesia

Kumpulan playlist IPTV channel Indonesia dalam format M3U.  
Playlist ini dapat digunakan langsung di aplikasi pemutar media yang mendukung format M3U seperti **VLC**, **Kodi**, **Smart TV**, maupun **Android IPTV Player**.

## Cara Menggunakan

### 1. Melalui VLC Media Player
- **Copy link berikut** ke clipboard:
  [IPTV](https://raw.githubusercontent.com/mubin-asrori/iptv-indonesia/main/playlist.m3u)
- Buka **VLC**, pilih menu `Media > Open Network Stream`.
- Paste link di atas ke kolom URL, klik **Play**.
- Daftar channel akan muncul dan bisa langsung dipilih.

### 2. Di Aplikasi Android IPTV
- Salin link playlist di atas ke aplikasi IPTV seperti **IPTV Smarters**, **TiviMate**, dll.
- Channel akan tampil otomatis.

### 3. Di Smart TV
- Cari aplikasi IPTV yang mendukung link M3U.
- Masukkan link playlist di atas pada pengaturan playlist.

## Tentang Playlist

- Berisi ratusan channel TV Indonesia, baik nasional maupun lokal.
- Format M3U dengan metadata yang kompatibel untuk EPG (Electronic Program Guide).
- [Geo-blocked] = Channel hanya dapat diakses dari wilayah Indonesia atau provider terkait.
- [Not 24/7] = Channel tidak selalu online.
